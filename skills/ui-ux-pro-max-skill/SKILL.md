## 🧠 161 Reasoning Rules

Unlike template-based generators, this skill **reasons** about design:

```
Input: "Serenity Spa" → 
  Analysis: wellness, calm, natural
  → Primary: Teal (#0F766E)
  → Secondary: Amber (#F59E0B)
  → Font: Playfair Display (heading) + Inter (body)
  → Components: Cards, Buttons, Forms (rounded, soft)
```
