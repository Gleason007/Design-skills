# taste-skill

> The Anti-Slop Frontend Framework for AI Agents.

## Overview

taste-skill upgrades AI-built interfaces with real design craft. Stronger layouts, better typography, purposeful motion, and proper spacing instead of boilerplate-looking UIs.

## 14 Skills

| Skill | Style |
|-------|-------|
| **taste-skill-v1** | Base design enhancement |
| **taste-skill** | Full design enhancement |
| **minimalist-skill** | Minimalist design |
| **brutalist-skill** | Brutalist design |
| **soft-skill** | Soft/rounded design |
| **redesign-skill** | Redesign existing work |
| **imagegen-frontend-web** | Web reference image generation |
| **imagegen-frontend-mobile** | Mobile reference image generation |
| **brandkit** | Brand kit generation |
| **image-to-code-skill** | Image → code conversion |
| **stitch-skill** | Google Stitch integration |
| **gpt-tasteskill** | ChatGPT version |

## Usage

```bash
# Enhance design quality
taste-skill

# Specific style
minimalist-skill
brutalist-skill
soft-skill

# Generate reference images
imagegen-frontend-web
```
